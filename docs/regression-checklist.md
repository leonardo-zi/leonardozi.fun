# 手动回归清单（路由与布局重构后）

在发布前快速过一遍：

1. **冷启动**：`/`，作品网格与首屏预加载、入场动画（Framer stagger）。
2. **作品详情**：点击卡片进入 `/work/:id`，返回回 `/`；从外链带旧参数 `/?work=1` 应重定向到 `/work/1`。
3. **路由切换**：`/music` ↔ `/`，列表正常；从音乐回作品时滚顶、卡片 remount 无空白卡。
4. **Lenis**：主列平滑滚动；侧栏/抽屉内滚动不锁死。
5. **侧栏**：桌面 `NavLink` 高亮与当前路径一致；移动端抽屉选分区后关闭。
6. **语言**：CN/EN 切换、`localStorage` 持久化；详情页与作品卡 copy 一致。
7. **断点**：`>1230` 双列作品；`≤1230` 穿插单列；`≤800` 顶栏/脚注布局与语言条位置。
8. **控制台**：无 GSAP/动效/React 报错。
